# **Nexus** | minecraft launcher

a simple, lightweight tool designed to help you switch between versions and keep your worlds safe.

### **features**
* **version switcher:** easily browse and install different versions of the game.
* **world backups:** save your progress with a single click before making changes.
* **client archive:** quick links to popular community tools and external clients.
* **smooth ui:** a clean, dark-themed interface with simple navigation.

---

### **how to install it**
1. **go [here.](https://github.com/fastmaxcode/nexus-launcher/releases/download/VersionSwitcher/Nexus.1.0.Beta.py)**

---

### **how to use it**

1.  **install python** make sure you have python 3.x installed on your system. you can download it from [python.org](https://www.python.org/).

2.  **get the requirements** this launcher uses a few extra tools to look and work great. open your terminal or command prompt and run:
    ```bash
    pip install customtkinter requests
    ```

3.  **run the launcher** download the script and run it using:
    ```bash
    python nexus_launcher.py
    ```

4.  **choose your version** once the window opens, just click on a version in the list. **N E X U S** will download the setup file and open it for you.

---

### **important notes**
* **backups:** it’s always a good idea to use the **backup worlds** button before switching versions to keep your data safe.
* **file location:** the launcher saves setup files in the same folder where the script is running.
* **system:** currently designed for windows users playing the bedrock edition.
